# Mosibello.com | Marketing

## Installing

```bash
# From Source
git clone https://github.com/mosibello/mosibello-marketing.git
cd mosibello-marketing
```

then install dependencies

```bash
  npm install
```

and run locally

```bash
  npm run dev
```

## Extra options

```bash
# Run a production build, locally
npm run build

# Serve a production build, locally
npm start
```
